# MODULE_INSTALLATION_PROGRES_TRACKING

Put the GeoJSON files for this module in this folder.

Expected files (default config):
- full.geojson
- string_text.geojson (optional)
- inv_id.geojson
- inv_box.geojson
- link (optional)

If you change filenames, also update the config in `src/modules/moduleConfigs.js`.
