# DC TERMINATION & TESTING PROGRESS

Place the dataset files for this module in this folder.

Expected files (same naming conventions as other modules):
- `full.geojson`
- `string_text.geojson`
- `inv_id.geojson`
- `lv_box.geojson` (optional)
- `boundry.geojson` (optional)
- `link` (optional text file containing a DWG/Drive URL)

After adding files, run the app and select **DC Termination & Testing Progress** from the hamburger menu.
